# Tetris AI
## Overview
Classic game of Tetris following the [Official Tetris Guidelines](https://tetris.fandom.com/wiki/Tetris_Guideline). Includes a greedy AI that evaluates the game board by:

- Number of holes
- Height of the stack
- Bumpiness
- Drop height
- Lines cleared
